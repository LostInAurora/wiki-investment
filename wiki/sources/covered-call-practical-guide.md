# 备兑看涨期权实战指南：股票收租与盈利进阶技巧

**Source:** 备兑看涨期权实战指南：股票收租与盈利进阶技巧.md
**Date ingested:** 2026-04-24
**Type:** guide

## Summary

该来源围绕 [[Covered Call]] 展开，核心观点是：对于已经持有正股且愿意在目标价格卖出的投资者，备兑看涨期权可以把持仓转化为持续的权利金收入。来源强调，执行前提不是“多赚一点权利金”，而是先确认自己确实愿意在行权价完成卖出。

来源提出两条关键纪律：一是优先在正股处于盈利状态时卖出 [[Covered Call]]，避免为了收取权利金而锁定本金亏损；二是更偏好在 [[RSI]] 高位、价格接近或突破 [[布林带]] 上轨时执行，以提高权利金并降低在低位卖飞的概率。文中还用 [[Walmart]]、[[Amazon]] 和 [[QBTS]] 作为案例，说明不同价格位置下应选用 [[Covered Call]] 还是 [[Sell Put]]。

## Key Claims

- [[Covered Call]] 适用于已有正股、且愿意在行权价卖出的持仓者。
- 执行前最好保证正股处于盈利状态。
- 更适合在 [[RSI]] 接近或高于 70、价格接近 [[布林带]] 上轨时卖出。
- 对不愿卖出的核心资产应谨慎使用甚至不使用 [[Covered Call]]。
- 30 至 45 天期限的合约更便于展期和风险管理。
- 处于严重超卖区的标的更适合 [[Sell Put]]。

## Entities Mentioned

- [[Walmart]] - 作为高位卖出 [[Covered Call]] 的示例。
- [[Amazon]] - 作为超卖状态下更适合 [[Sell Put]] 的示例。
- [[QBTS]] - 作为反复卖出和买回 [[Covered Call]] 以降低成本的示例。

## Concepts Covered

- [[Covered Call]] - 该来源的核心期权策略。
- [[Sell Put]] - 用于与 [[Covered Call]] 对比的替代策略。
- [[Wheel Strategy]] - 由卖 put 和卖 call 组成的闭环。
- [[RSI]] - 用于判断超买和超卖状态。
- [[布林带]] - 用于识别高位和波动区间位置。
