# 定投底仓

## Summary

[[定投底仓]] 是当前来源中承担长期复利、稳定资产增值和风险锚定的账户部分。作者把它看作账户的核心层，通常由 [[QQQM]]、[[VOO]] 和精选个股构成，并通过年龄公式决定其总体占比。

## Key Notes

- 比例由 [[仓位配比系统]] 决定。
- 内部结构强调成长暴露、宽基暴露和精选个股的平衡。
- 单一个股仓位不应过度集中。
- 在 [[现金流瀑布]] 规则下，它是期权盈利最终流入的主要目标。
